# SimpleServerGo

Simple Server with Golang

1. Clone / Download File
2. Open Terminal
3. Open File with command "cd SimpleServerGo"
4. Run File with command "go run main.go"
5. Open Browser
6. Open http://localhost:8080/

List Endpoints :
1. "/" (Main Page)
2. "/hello" (Hello Server)
3. "/form" (Form)
